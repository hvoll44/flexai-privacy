FlexAI Flutter

Your AI-Powered Personal Trainer

Project Overview

FlexAI Flutter is a fitness application that leverages AI to create personalized workout programs. The app allows users to generate, manage, and track workout routines tailored to their specific fitness goals and preferences.

Key Features

  • AI-Powered Workout Generation: Create personalized workout programs using OpenAI's API
  • Program Management: Create, view, edit, and delete workout programs
  • Workout Tracking: Track sets, reps, and weights for each exercise
  • User Preferences: Customize workouts based on fitness level, goals, available equipment, and schedule
